# Linux Production Troubleshooting Lab

This project contains real Linux troubleshooting scenarios created and fixed in a lab environment.

## Purpose

The goal of this project is to practice production-style Linux troubleshooting using real commands, logs, service checks, networking checks, firewall checks, and permission debugging.

## Topics Covered

- Nginx troubleshooting
- SSH troubleshooting
- DNS and /etc/hosts issues
- Firewall issues
- Disk, memory, and CPU troubleshooting
- Mounting and fstab issues
- Log permission issues

## Troubleshooting Format

Each issue is documented using this format:

1. Issue
2. Symptoms
3. Commands Used
4. Observation
5. Root Cause
6. Fix
7. Verification
8. Interview Explanation
